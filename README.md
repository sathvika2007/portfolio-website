# Responsive Portfolio Website

## Task
Frontend Development Internship – Alfido Tech

---

## Objective
The objective of this project is to create a responsive personal portfolio website using HTML and CSS, showcasing personal, academic, and technical information effectively.

---

## Tools & Technologies Used
- HTML5
- CSS3
- GitHub
- GitHub Pages
- VS Code

---

## Project Description
I developed a responsive personal portfolio website as part of my Frontend Development Internship at Alfido Tech.

The website showcases my profile, education, skills, projects, internship experience, and contact details in a clean and structured layout.

The portfolio includes sections such as:
- Home
- About
- Education
- Skills
- Projects
- Internships
- Contact

The website is designed with a modern user interface and responsive layout for better user experience across devices.

---

## Features
- Responsive design
- Smooth navigation bar
- Structured layout
- Personal profile section
- Skills and projects section
- Internship section
- GitHub and LinkedIn integration
